# Discord-MC
A simple script made to link Minecraft and Discord servers.
## Getting Started
### Compatibility
Latest Nodejs
### Installing
> Clone the Project
```
git clone https://github.com/TotherMeteor/Discord-MC.git
```
> Prerequisites
```js
npm install discordjs
npm install mineflayer
```
> Config
Add your login credentials to the config.js
### Running
```js
nodejs index.js
```
